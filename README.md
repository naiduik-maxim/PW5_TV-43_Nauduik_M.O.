# **Практична робота №3: HTML-форми та серверна обробка (Node.js \+ Express)**

## **Мета роботи**

Розробка клієнт-серверного веб\-додатка для збору та обліку результатів технічної діагностики обладнання. Практичне закріплення навичок створення інтерактивних HTML-форм, налаштування маршрутизації та обробки HTTP-запитів (GET, POST, DELETE) за допомогою Node.js та фреймворку Express, а також реалізація збереження даних на сервері у форматі JSON із дворівневою валідацією (клієнт/сервер).

## **Інструкція із запуску**

Оскільки цей проєкт містить серверну частину (Backend), для його повноцінної роботи потрібне середовище виконання Node.js:

1. Переконайтеся, що на вашому комп'ютері встановлено **Node.js**.  
2. Відкрийте термінал (або командний рядок) безпосередньо в кореневій папці проєкту.

Встановіть необхідні модулі (фреймворк Express), виконавши команду:  

Bash  
npm install
npm install express

Запустіть локальний сервер командою:
Bash  
node server.js
 
3. Дочекайтеся повідомлення `server run: http://localhost:3000`  
4.  у терміналі, після чого відкрийте веб\-браузер та перейдіть за адресою **http://localhost:3000**.
