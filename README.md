# **Практична робота №5: Клієнтський веб-додаток (JavaScript + API)**

## **Мета роботи**

Розробка клієнтської частини веб-застосунку для роботи з API (WebSocket) з можливістю завантаження та візуалізації даних системи моніторингу споживання промислового підприємства у реальному часі. Практичне закріплення навичок створення адаптивного інтерфейсу (Bootstrap 5, кастомний дизайн SCADA), ініціалізації та оновлення динамічних графіків (Chart.js) та забезпечення стабільного двостороннього з'єднання між клієнтом і сервером (Node.js + `ws`).

## **Інструкція із запуску**

Оскільки цей проєкт містить серверну частину (Backend) для генерації та трансляції даних через WebSocket, для його повноцінної роботи потрібне середовище виконання Node.js:

1. Переконайтеся, що на вашому комп'ютері встановлено **Node.js**.  
2. Відкрийте термінал (або командний рядок) безпосередньо в кореневій папці проєкту.

Встановіть необхідні модулі (бібліотеку для роботи з WebSocket), виконавши команди:
